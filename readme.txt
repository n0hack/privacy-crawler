사용법
1. 네이버 아이디, 비밀번호를 입력한 이후 계정 저장 버튼을 클릭합니다.
→ 최초 실행 시에만 필요한 작업이며, 이후 프로그램을 다시 켜면 저장된 정보로 자동 로그인합니다.
2. 계정 저장이 완료되었다면, 수집 시작 버튼을 클릭해서 진행합니다. (하단에서 진행 상황 확인 가능)
→ 블로거 리스트가 작성된 텍스트 파일을 읽은 후 진행됩니다.
3. 수집이 모두 완료된 경우, 결과를 저장하면 됩니다. (진행 도중 저장 X / 텍스트로 저장 됩니다.)

유의사항
1. 대형 카페를 위주로 정보를 불러오므로 몇개 카페의 가입이 필요합니다. (필수. 중고나라)
→ 네이버의 경우는 카페 정보 열람 시 가입이 필수이므로 중고나라 카페는 가입된 아이디여야 합니다.
2. 프로그램을 통한 정보 수집은 개인정보 보호법을 따르지 않기 때문에 주의를 요합니다.
3. 개발자의 요청에 따라 본 프로그램은 재배포를 절대 금지합니다.
